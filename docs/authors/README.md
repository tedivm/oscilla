# Content Author Documentation

Welcome to the content author documentation! This directory contains guides for creating and maintaining game content for oscilla using YAML manifests.

If you are looking to work on the engine itself, see the [Developer Documentation](../dev/README.md) instead.

## Getting Started

New to content authoring? Start here:

1. **[Content Authoring Guide](./content-authoring.md)** — Manifest format, every kind with field reference and examples, conditions, steps, effects, the `validate` command, and common mistakes
2. **[Skills and Buffs](./skills.md)** — Skill and Buff manifest reference, equipment skill/buff grants, enemy skills, CharacterConfig skill resources, and the buff variables system
3. **[Pronouns](./pronouns.md)** — Pronoun placeholder reference, verb agreement, capitalisation, custom pronoun sets via `CharacterConfig`
4. **[Passive Effects](./passive-effects.md)** — Passive effect YAML structure, condition restrictions, stat modifiers, and skill grants
